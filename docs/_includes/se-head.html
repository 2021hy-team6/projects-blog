분리수거장에서 만나는 재활용 파트너

Recycling Assistant

repository / paper / demo video

Recycling Assistant는 인식한 영상을 바탕으로
정확한 분리수거 요령을 설명하는 스마트 IoT 가전입니다.

  • 사물의 구성요소를 자동으로 인식하는 AI
  • 다양한 카메라와 연결 가능한 IoT
  • 재활용 현황을 실시간으로 파악하는 Dashboard

자원순환 X AI

스마트한 분리배출의 시작

우리가 하는 분리수거는 모두 재활용될까요?
재활용인 척하는 일반쓰레기와, 혼합구성된 재활용품은
자원순환의 실천을 방해하고 있습니다.*
분리수거 하는 여러분의 곁에서 인공지능이 돕겠습니다.

* 출처 : Kwaste 전국 폐기물발생 및 처리현황

실시간 영상인식 AI

1초 안에 설명해드립니다*

카메라에 물건을 보여주기만 한다면
재활용 가능한 구성요소를 단숨에 찾아냅니다.
대분류 10종, 소분류 60여종의 재활용품이 식별 가능한 AI는
우리가 몰랐을 수도 있는 분리배출 요령을 제안합니다.

* 이미지 프레임 별 평균 인식속도 : 0.29초 (자체 테스트 결과)

다양한 입력방식

원하는 카메라 마음대로

USB Webcam, IP Camera, 스마트폰*까지 모두 지원합니다.
아파트, 회사, 다세대 주택, 셰어하우스 어디든지
상황과 필요에 맞게 Recycling Assistant를 사용하면 됩니다.

* IP 카메라를 지원하는 어플리케이션 설치 후 활용가능

재활용 빅데이터

분리수거 현황을 한눈에

실시간으로 항목별 분리배출 현황을 조회할 수 있습니다.
언제 어떤 물건이 얼마나 재활용 되는지에 관한 데이터로
일상생활과 재활용에 대한 새로운 인사이트를 찾을 수 있습니다.
